<h1 align="center">GraphQL Backend (Spring Boot)</h1>

Description:
Parallel GraphQL API over the same PostgreSQL schema as REST. Exposes typed queries and mutations to fetch nested data (e.g., product + reviews) in one request, and a checkout mutation that creates the order and clears the cart.

Run locally:

Open the project in your IDE and Run as Java Application (Spring Boot main class), or:

mvn spring-boot:run
